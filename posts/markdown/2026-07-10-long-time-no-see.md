# Long time no see

## Intro

Now I got to admit that I didn't do much work at all. No visible progress. But it's okay because I've progressed philisophically in a sense.

## Goals

I feel like I've just been doing too much robotics. I don't feel like that is the goal of doing programming for me. I don't feel like (at least right now) as a solo developer I should put my eggs in one basket just yet. What I mean by that is, I don't want to *just* do robotics, or *just* focus on web developement. My goal for programming is to help people, even just a little. This can't really be done by *just* one discipline, it has to be flexible. Sometimes I want to create an app, in that case I can do web development to make it accesible. If I want to make a physical product to help people physically, then I would make a robot. TL;DR: I don't care about speciality within programming disciplines anymore, I will make products however format they come out.
