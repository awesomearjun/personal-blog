# Blog + Hustle

## Intro

This week I tried a little more different stuff, I worked on more web-devy projects — STILL NOT A WEB DEVELOPER THOUGH DON'T GET IT TWISTED. I made improvements to this blog, with a hamburger menu for mobile and all the links in the navbar actually have pages now. There's the about page, projects page — which I still need to make images work, and a contact page.

## Blog

How did I do all this in a week? Well I didn't do it, I actually installed Cursor and told it to code for me. It seems bad coming from NeoVim to Cursor, but I just could not for the life of me figure out how to put AI in NeoVim. I can just get a lot more work done with AI and agents, and if I just refuse to use it then I would fall behind. I can code entire website prototypes in a matter of minutes — and good ones too.<br><br>

Cursor really got better from last time I used it, a couple years ago. Seems like you can get more quality for 💩 prompts. This time, my prompt was really vague, just: "look at the navbar in this file, and code pages for it keeping in mind styles in this file". But it actually worked, and I'm super happy with the design. Now if I want to apply for an internship or a job someday, I actually have something to refer to. I admit, this was really vibe-codey, but the next project was genuinely planned and executed. I think the last time I used it, the code just never ran, but this recent time it seems that Cursor can fix it's own code which is mind-boggling.

## Hustle

So I wanted to make a side hustle; use my skills to get dollar bills; kwim twin? As I was searching for a project idea, which always takes stupidly long, I stumbled on one. It wasn't the first of it's kind, but it is hella usable. I wanted to create a YouTube thumbnail a/b testing software with an AI chatbot; it turned out great. This wasn't as vibe-codey as the blog, I did have to be super specific, and learned some things along the way. To make this, I explored:

- Planning in Cursor, stupidly easy because there's just a dropdown to switch to a planning model, and it creates a super nice plan as if you were in a company with big compound words no one actually knows
- Agents, it took a little while to code, & i only used one, but I really did feel like a Senior developer looking at all the files the AI commited while pretending to have a negative attitude
- New libraries, such as NX which to this day I don't know what it does but the AI used it and it sounds cool so overall it gets a 👍/10
